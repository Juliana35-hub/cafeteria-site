# ☕ Encanto do Grão - Cafeteria Artesanal

Um site institucional premium para uma cafeteria, desenvolvido com foco em design sofisticado, responsividade, microinterações e alta performance.

## 🎯 Objetivo
O projeto foi criado para demonstrar habilidades em **Frontend Development**, focando em uma interface refinada que prioriza a experiência do usuário (UX), imersão visual e a conversão (pedidos diretos via WhatsApp).

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e otimizada para SEO.
- **CSS3**: Design responsivo (Mobile-First), CSS Grid, Flexbox, variáveis nativas e micro-animações avançadas.
- **JavaScript (Vanilla)**: Lógica do carrossel de depoimentos, barra de progresso de scroll, navbar mobile e Intersection Observer para animações de revelação (scroll).

## 🛠️ Funcionalidades
- [x] **Layout Fluido e Responsivo**: Adaptação perfeita e testada para smartphones, tablets e desktops.
- [x] **Animações "Reveal"**: Efeito elegante de fade-in e slide-up sincronizado ao scroll da página com atraso em cascata (stagger).
- [x] **Carrossel Interativo**: Seção dinâmica de depoimentos de clientes com touch/arrasto suave.
- [x] **Cardápio Estilizado**: Organizado em Grid com efeitos premium de hover (zoom sutil na imagem e sombra dinâmica).
- [x] **Barra de Progresso (Scroll)**: Indicador visual do progresso de leitura no topo da tela.
- [x] **Sticky Header**: Navegação translúcida com efeito blur que acompanha a rolagem.

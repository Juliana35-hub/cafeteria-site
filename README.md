# ☕ Café Aurora - Site Institucional

Um site institucional premium para uma cafeteria fictícia, desenvolvido com foco em design sofisticado, responsividade e alta performance.

## 🎯 Objetivo
O projeto foi criado para demonstrar habilidades em **Frontend Development**, focando em uma interface "chic minimalista" que prioriza a experiência do usuário (UX) e a conversão (contato via WhatsApp).

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e otimizada para SEO.
- **CSS3**: Design responsivo (Mobile-First), CSS Grid, Flexbox e animações customizadas.
- **JavaScript (Vanilla)**: Manipulação de DOM, Intersection Observer para animações de scroll e lógica de menu mobile.

## 🛠️ Funcionalidades
- [x] **Layout Responsivo**: Adaptável para dispositivos móveis, tablets e desktops.
- [x] **Sticky Header**: Navegação que se adapta ao scroll da página.
- [x] **Animações de Entrada**: Efeito de fade-in/slide-up conforme o usuário navega.
- [x] **Cardápio Interativo**: Organizado em Grid com efeitos de hover.
- [x] **Botão WhatsApp**: Link direto para atendimento rápido.
- [x] **Smooth Scroll**: Navegação suave entre as seções.
